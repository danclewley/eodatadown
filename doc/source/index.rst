Earth Observation Data Downloader (EODataDown)
==============================================

The Earth Observation Data Downloader (EODataDown) is a tool for automatically downloading and processing EO data to an analysis ready data product. This software forms a core component of a monitoring system based on EO data with user defined plugins used to implement the data analysis.


EODataDown is available to download from https://github.com/remotesensinginfo/eodatadown
